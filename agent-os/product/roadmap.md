---

## Version 1.0 — Process & Visibility

**Theme:** Establish the foundation. Make the invisible visible.

**Goal:** Replace spreadsheets and WhatsApp chaos with structured workflows.

### Core Deliverables

| Task | Description |
| --- | --- |
| Identity & Access Management | Role-based permissions and access controls |
| Entity Model | Three entities: Lead → Client → Case with clear conversion points |
| MS Dashboard | List of assigned clients with SLA timers. Filter by status, source, date. |
| Client Management | Create client, select source channel, enter identity. Support joint applications. |
| Eligibility Calculator | Input salary + liabilities. Auto-compute DBR, LTV, max loan amount. |
| Document Collection | Upload all document types. |
| Notes & Activity | Add notes (General, Quick actions). Unified activity timeline across entities. |
| Client Closure | Close clients through quick actions like declined and withdrawn. |
| PO Dashboard | Kanban board with case stages as columns. SLA timer per case. |
| Case Management | Create case from qualified client. Select bank, enter deal details. Move through stages. |
| Case Closure | Close case using quick actions, Declined, or Withdrawn. |
| Manager Dashboard (Basic) | Team workload, SLA health, reassign leads/cases. |
| Admin Settings | Users & roles, channels |
| Audit Logs | Immutable record of every state change. |
